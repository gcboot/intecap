🚀 INTECAP - Programación en Python
Este repositorio contiene una colección de ejercicios, guías y hojas de trabajo desarrolladas durante el curso de formación en INTECAP. El objetivo es documentar el progreso en el aprendizaje de lógica de programación, estructuras de datos y Programación Orientada a Objetos (POO).

📂 Contenido del Repositorio
El proyecto está organizado en secciones que cubren desde los fundamentos hasta conceptos avanzados:

Fundamentos de Lógica: Ejercicios con ciclos for y while (como el generador de boletos y acumuladores de precios).

Estructuras de Datos: Manejo de listas y colecciones de productos.

Manejo de Excepciones: Funciones robustas con bloques try-except para evitar errores en tiempo de ejecución.

Programación Orientada a Objetos (POO): * Clases y Objetos (Clase Libro).

Herencia (Clases Empleado y Gerente).

Encapsulamiento (Atributos privados y métodos de acceso).

Polimorfismo (Sobrescritura de métodos en subclases).

🛠️ Tecnologías Utilizadas
Lenguaje: Python 3.x

Entorno: VS Code / PyCharm

Paradigma: Programación Imperativa y Orientada a Objetos
